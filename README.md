# simple-procedural-map-generation-code-template-for-godot
This is a template for anyone wanting to do procedural map generation for their game
This is ment to be useds as a template. While it can be used as is, it's recomended to be modifyed to fit your needs
I will provide so free tiles made by me if you don't have tiles or you can check out https://itch.io/game-assets/free if you want better tiles
This can be used for non-commercial or commercial uses, but credits is appreciated
